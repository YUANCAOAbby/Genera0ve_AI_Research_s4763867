# Generative AI Research Project

This repository is used to organise research materials for the project **"Using Generative AI Tools - Boon or Bane"**.

## Folder Structure

- `01_Literature_Review`: articles, books, and other literature sources.
- `02_Quantitative_Analysis`: survey data, survey questions, and analysis files.
- `03_Qualitative_Analysis`: interview transcripts, interview protocols, and consent forms.
- `04_Drafts_and_Reports`: draft proposals, conference papers, and final reports.
- `05_Additional_Materials`: information sheets, photos, and other media files.

## Access Control

Sensitive files, such as consent forms, interview transcripts, raw survey data, and identifiable photos, should only be accessed by authorised team members.

## File Naming

Files should use clear names with dates or version numbers where needed.

Example: `SurveyData_Cleaned_2026-04-30_v1.csv`

## Collaboration

Team members should use clear commit messages when uploading or changing files.
